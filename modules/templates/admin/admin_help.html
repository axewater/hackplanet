Administrator Help

Initial Setup

  1. Ensure all environment variables are set correctly in your configuration file or deployment environment.
  2. Run database migrations to set up the required tables.
  3. Create an admin user account if not already done.
  4. Configure email settings for user registration and password reset functionality.

Admin Dashboard Overview

  • Server Settings: Configure global application settings.
  • Server Info: View system information and current configuration.
  • User Manager: Manage user accounts, roles, and permissions.
  • Whitelist: Manage email addresses allowed to register.
  • Invite System: Control user invitations.
  • Lab Manager: Create and manage hacking labs.
  • Host Manager: Manage individual hosts within labs.
  • Challenge Manager: Create and manage hacking challenges.
  • Quiz Manager: Create and manage quizzes.

Setting Up a New Lab

  1. Go to the Lab Manager.
  2. Click "Add New Lab".
  3. Fill in the lab details including name, description, and any associated files.
  4. Supply the VPN server name, which should be the Azure name (not the host name). This allows for VPN gateway management from within the application.
  5. Save the lab.
  6. Use the Host Manager to add hosts to the lab.

Creating a New Challenge

  1. Navigate to the Challenge Manager.
  2. Click "Add New Challenge".
  3. Enter the challenge details, including name, description, and flag.
  4. Set the point value for the challenge.
  5. Save the challenge.

Setting Up a Quiz

  1. Go to the Quiz Manager.
  2. Click "Add New Quiz".
  3. Enter the quiz title and description.
  4. Set the minimum passing score.
  5. After saving the quiz, add questions using the "Add Question" button.
  6. For each question, provide the question text, options, correct answer, and point value.

For more detailed information or troubleshooting, please refer to the full documentation or contact the system administrator.
