How to Create a Theme for HackPlanet

Creating a theme for HackPlanet is easy. Follow these steps:

  1. Create a new directory for your theme.
  2. Inside this directory, create a file named theme.json with the following structure: 
    {
    "name": "Your Theme Name",
    "description": "A brief description of your theme",
    "author": "Your Name",
    "release_date": "YYYY-MM-DD"
}
  3. Create a subdirectory named css.
  4. Inside the css directory, create your CSS files. The main CSS file should be named style.css.
  5. Zip the entire theme directory.
  6. Upload the zip file using the theme upload form.

That's it! Your theme should now be available for use in HackPlanet.
