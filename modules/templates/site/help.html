HackPlanet.EU Help Center

Getting Started

  • Each LAB is a separate network with machines you can hack.
  • To access each LAB, download and use the provided OpenVPN connection file.
  • Submit flags to check your progress and earn points.
  • Check the leaderboards to see how you rank among other participants.
  • Manage VMs by starting and stopping them, including the VPN server.

Rules of Engagement

  • Brute force password cracking is never the intended solution. You may try, but it's likely not the correct path.
  • Do not attack the VPN server or the HackPlanet.eu webserver.
  • Do not attack or disrupt other participants.
  • Once you gain root access, do not delete flags. Please clean up exploit-related files when finished to prevent spoilers.
  • If a host becomes unresponsive, contact support for a reboot/rebuild.

Technical Details

  • The VPN uses a TUN interface, making it incompatible with nmap under Windows.
  • A Kali Linux host is available inside each LAB network for all HackPlanet users.
  • The VPN server should always be running if you want to connect to the lab.
