Getting Started :


- Each LAB is a seperate network with machines you can hack. To access each LAB, an OpenVPN connection file can be downloaded
- Submit your flags to check your progress and earn points
- Check the leaderboards to see how you rank among other participants

Rules of Engagement:


- Brute force password cracking is never (intended) way to the goal. You are allowed to try, but you are probably on the wrong path.
- Do not attack the VPN server or the Hackplanet.eu webserver.
- Do not attack or disrupt other participants.
- Once you get root, do not delete flags. It is appreciated if you cleanup exploit related files when you are done. (To prevent spoilers) - If a host seems to have stopped responding, contact support for a reboot/rebuild


Technical Details:

- The VPN is using a TUN interface, this makes it incompatible with nmap under Windows. - There is a KALI host available inside each LAB network; for all HackPlanet users.
