Getting Started with HackPlanet.EU

Welcome to HackPlanet.EU

HackPlanet.EU is a cybersecurity learning platform designed to help you develop and test your hacking skills in a safe, legal environment.

Prerequisites

  • A machine or virtual machine running Kali Linux
  • OpenVPN client installed (for Hacking Labs access)
  • Basic understanding of cybersecurity concepts

Getting Started Steps

  • Hacking Labs:
    • Install OpenVPN on your Kali Linux system
    • Download the OVPN file from the Hacking Labs page
    • Connect to the lab using OpenVPN
    • Start exploring and hacking the lab machines
  • Crypto Challenges:
    • Access the /ctf/challenges page
    • No VPN connection required
    • Download challenge files if provided
    • Submit flags to earn points
  • Quizzes:
    • One-time attempts only
    • Points awarded for correct answers
    • Review explanations after completion
Note: You can participate in Crypto Challenges and Quizzes without VPN access. Hacking Labs require VPN connection.
