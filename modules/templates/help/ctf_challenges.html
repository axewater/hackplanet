CTF Challenges

Understanding CTF Challenges

Capture The Flag (CTF) challenges test your ability to find and obtain flags hidden within target systems. Each challenge requires different skills and approaches.

Flag Locations

In our hacking labs, there are two types of flags to find on each target host:

  • User Flag: Located at:
    • Linux: /home/user/userflag.txt
    • Windows: %username%/desktop/userflag.txt
  • Root Flag: Located at:
    • Linux: /root/rootflag.txt
    • Windows: users\administrator\desktop\rootflag.txt

Flag Submission

  • Submit flags through the web interface on the Hacking Labs page
  • Each host has different point values for user and root flags
  • Points are awarded immediately upon successful submission
  • Higher difficulty hosts offer more points

Challenge Types

  • Web Exploitation
  • Cryptography
  • Reverse Engineering
  • Network Security
  • Forensics
Tip: Always start with enumeration and information gathering before attempting to obtain flags.
